# Fake News Classifier (NumPy-only Neural Network)

This project implements a fake news classifier using:
- custom tokenization
- unigram, bigram, trigram vectorizer
- manual neural network (NumPy only)
- Adam optimizer

## Dataset

The dataset is not included in the repository due to size limits.

Download the dataset from the Kaggle link:
https://www.kaggle.com/datasets/clmentbisaillon/fake-and-real-news-dataset

Or download the preprocessed version:
(data.rar included in this repo)

Extract `data.rar` and place the contents inside:

fake-news-classifier/data/


## Run main
python src/main.py
